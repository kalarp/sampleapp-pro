Quickstart Guide

1. Installation

pip install sampleapp-pro
npm install @sampleapp-pro/core

2. Initial Configuration

Create a configuration file sampleapp.config.json:

{
    "data_sources": ["postgresql", "mongodb"],
    "transformation_mode": "realtime",
    "log_level": "info"
}

3. Create Your First Pipeline

from sampleapp_pro import Pipeline

pipeline = Pipeline()
pipeline.source('postgresql://mydb')
    .transform(clean_data)
    .destination('mongodb://target')
    .run()
